美容サロン向け 在庫・発注管理ソリューション

システム概要

各店舗からの発注を本部で一元管理し、効率的な購入作業をサポートするシステムです。

主な特徴:
  • 店舗ごとの専用発注URL（ログイン不要でシンプル）
  • 商品別・メーカー別の発注集計
  • 担当者別の購入リスト
  • 本部在庫の自動判定による担当振り分け

購入リスト（重要）

浅野さん・金本さんがECサイトで購入する際に使う画面です

各店舗からの発注を商品別に集計し、メーカー順に表示します。

担当者の自動判定ロジック

条件 担当者
本部在庫がある 浅野さん（商品の担当設定に関わらず）
本部在庫がない 商品マスターの担当者設定に従う

使い方

  1. 担当者フィルターで自分の担当分を絞り込む
  2. メーカー別にECサイトで購入
  3. 購入したらチェックボックスにチェック
  4. 進捗が画面上部に表示される

店舗発注画面

各店舗のスタッフが使う発注入力画面です。ログイン不要でシンプルに使えます。

発注URL

https://[ドメイン]/store/[店舗コード]

※店舗マスター画面でURLをコピーできます

操作手順

  1. 商品一覧から発注したい商品をタップ
  2. 数量を調整（＋/−ボタン）
  3. 「発注を送信」ボタンを押す
  4. 発注番号が表示されたら完了

運用フロー

[店舗] [本部] │ │ │ 1. 発注画面で商品を選択 │ │ 2. 発注を送信 │ │ ───────────────────────────→ │ │ │ 3. 発注管理で確認 │ │ 4. 購入リストで担当別に確認 │ │ 5. ECサイトで購入 │ │ 6. 商品到着後、在庫更新 │ ←─────────────────────────── │ │ 7. 店舗へ発送 │

商品マスターの担当者設定

各商品にデフォルトの発注担当者を設定できます。

設定方法:
  1. 商品マスター画面を開く
  2. 商品を新規登録または編集
  3. 「担当者（ラベル）」で担当者を選択
  4. 保存

※本部在庫がある場合は、この設定に関わらず浅野さんが担当になります。
