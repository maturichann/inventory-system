# 在庫管理・発注システム 機能説明書

---

## システム概要

美容サロン向けの在庫管理・発注システムです。
各店舗からの発注を本部で一元管理し、効率的な購入作業をサポートします。

---

## 画面構成

### 1. ダッシュボード（トップページ）
システム全体の状況を一目で確認できます。

- **発注待ち件数**: 未処理の発注数
- **本日の発注**: 当日受け付けた発注数
- **在庫アラート**: 閾値を下回った商品数
- **登録商品数**: マスター登録されている商品の総数

---

### 2. 商品マスター `/products`
商品情報を管理します。

**登録項目:**
- 商品コード（必須）
- 商品名（必須）
- カテゴリ
- メーカー
- 階層分類（最大8階層）
- 単価・原価
- 仕入れ先
- **担当者ラベル**（浅野/金本など）
- メモ

**担当者ラベルについて:**
- 各商品にデフォルトの発注担当者を設定可能
- 購入リストで担当者別にフィルタリングできます

---

### 3. メーカーマスター `/makers`
仕入れ先メーカーを管理します。

**登録項目:**
- グループコード
- メーカー名
- 発注区分
- 最低発注数

---

### 4. 店舗マスター `/stores`
各店舗を管理します。

**登録項目:**
- 店舗コード
- 店舗名

**発注URL:**
店舗ごとに専用の発注画面URLが自動生成されます。
```
https://[ドメイン]/store/[店舗コード]
```
URLをコピーして各店舗に共有してください。

---

### 5. 担当者マスター `/staff`
発注担当者を管理します。

**登録項目:**
- 名前
- メールアドレス
- 役割（管理者/スタッフ）

---

### 6. 本部在庫 `/inventory`
本部の在庫数を管理します。

**機能:**
- 在庫数の登録・更新
- 閾値（アラート基準）の設定
- 在庫数が閾値以下の商品はハイライト表示

---

### 7. 発注管理 `/orders`
店舗からの発注を一覧で確認・処理します。

**ステータス:**
- `pending`: 発注待ち
- `processing`: 処理中
- `completed`: 完了
- `cancelled`: キャンセル

**機能:**
- 発注内容の確認
- ステータスの更新
- 担当者の割り当て

---

### 8. 購入リスト `/purchasing` ★重要

**浅野さん・金本さんがECサイトで購入する際に使う画面です。**

**機能:**
- 各店舗からの発注を**商品別に集計**
- **メーカー別にグループ化**して表示
- **担当者別にフィルター**可能

**担当者の自動判定ロジック:**
| 条件 | 担当者 |
|------|--------|
| 本部在庫がある | **浅野さん**（商品の担当設定に関わらず） |
| 本部在庫がない | **商品マスターの担当者設定**に従う |

**表示項目:**
- 担当者ラベル（浅野/金本/未設定）
- 商品コード
- 商品名
- カテゴリ
- 合計数量（全店舗合計）
- 本部在庫数
- 店舗内訳（どの店舗が何個必要か）

**使い方:**
1. 担当者フィルターで自分の担当分を絞り込む
2. メーカー別にECサイトで購入
3. 購入したらチェックボックスにチェック
4. 進捗が画面上部に表示される

---

### 9. 在庫アラート `/alerts`
在庫が閾値を下回った商品の一覧です。
早めの発注が必要な商品を確認できます。

---

### 10. 店舗発注画面 `/store/[店舗コード]`

**各店舗のスタッフが使う発注入力画面です。**

**特徴:**
- シンプルな画面（ログイン不要）
- 店舗ごとに専用URL
- 商品をクリックしてカートに追加
- 数量を調整して発注送信

**操作手順:**
1. 商品一覧から発注したい商品をタップ
2. 数量を調整（＋/−ボタン）
3. 「発注を送信」ボタンを押す
4. 発注番号が表示されたら完了

---

## 運用フロー

```
[店舗]                    [本部]
  │                         │
  │ 1. 発注画面で商品を選択    │
  │ 2. 発注を送信            │
  │─────────────────────────→│
  │                         │ 3. 発注管理で確認
  │                         │ 4. 購入リストで担当別に確認
  │                         │ 5. ECサイトで購入
  │                         │ 6. 商品到着後、在庫更新
  │←─────────────────────────│
  │ 7. 店舗へ発送             │
```
